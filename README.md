# Voice-Recorder
This is a simple voice recorder. It records high quality(3GP).  Recordings are stored on the SD card in the "voice reorder" folder. Main features ★ Record all your audios into 3GP, which can save a lot of space for you. You can now record for hundreds of hours! ★ Easy to manage all the audio records. Long press on the files to delete or rename as needed. ★ Play the audio recordings with the help of built in media player with super ease.
